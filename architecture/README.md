# INDmoney RAG Chatbot Architecture

This folder contains high-level documentation of the system. For the actual code, please refer to the `phaseX` directories.

## Overview
The project is built on a 5-phase modular architecture:

- **[Phase 1](./../phase1)**: Data Ingestion (Scraping).
- **[Phase 2](./../phase2)**: Embedding & Vector Store.
- **[Phase 3](./../phase3)**: Backend Services (RAG & API).
- **[Phase 4](./../phase4)**: Frontend Development (Tiny UI).
- **[Phase 5](./../phase5)**: Data Scheduler (Orchestrator).

Detailed documentation can be found in [overview.md](./overview.md).
